# TOEIC Learning Platform

A self-hosted web application for TOEIC preparation, featuring:
- Exam simulation (7 parts)
- Review mode
- Progress tracking
- Audio and image support

## Tech Stack
- **Backend**: Python FastAPI
- **Frontend**: React (Vite) + Tailwind CSS

## Usage
1. Place test data in `data/tests/`.
2. Start backend: `cd backend && uvicorn main:app --reload`
3. Start frontend: `cd frontend && npm run dev`
